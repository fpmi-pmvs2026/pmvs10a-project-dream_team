# Проект «Мои Финансы»

Добро пожаловать в официальную документацию мобильного приложения для управления личными финансами. Проект разработан командой **dream_team**.

## Краткое описание задачи
Цель проекта — создание удобного Android-приложения для фиксации ежедневных расходов с поддержкой мультивалютности (BYN/USD). Приложение автоматически конвертирует затраты, используя актуальные курсы валют, и предоставляет наглядную аналитику по категориям.
