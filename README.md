# Мои Финансы (My Finance)

## Description
«Мои Финансы» — это мобильное Android-приложение для удобного учета и анализа ежедневных расходов. Приложение позволяет пользователям фиксировать траты с указанием категорий и получать актуальные курсы валют через стороннее API. Реализована наглядная страница статистики с подсчетом общих затрат и детальной разбивкой по каждой категории. Продукт построен на архитектуре MVVM и поддерживает локализацию, а также смену светлой и темной тем оформления.

## Installation
Для локального развертывания проекта выполните следующие шаги:
1. Убедитесь, что у вас установлена актуальная версия Android Studio.
2. Склонируйте репозиторий на свой компьютер, используя ссылку: https://github.com/fpmi-pmvs2026/pmvs10a-project-dream_team
3. Откройте проект в Android Studio (File -> Open и выберите папку проекта).
4. Дождитесь завершения синхронизации Gradle (Gradle Sync).
5. Нажмите кнопку Run (зеленый треугольник) на панели инструментов или используйте сочетание клавиш Shift + F10, чтобы запустить приложение на эмуляторе или подключенном физическом устройстве.

## Usage
После успешной установки и запуска приложения:
1. **Главная страница:** Введите название покупки, сумму, выберите категорию из выпадающего списка и нажмите «Сохранить». Запись мгновенно отобразится в списке ниже.
2. **Статистика:** Перейдите на вкладку «Статистика» в нижнем меню, чтобы увидеть общую сумму всех расходов и детализированные карточки по категориям.
3. **Настройки:** Перейдите на вкладку «Настройки» для переключения темной/светлой темы или смены языка приложения (Русский/English).
