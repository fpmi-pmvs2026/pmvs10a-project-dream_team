# Мои Финансы (My Finance) — dream_team

## Описание
**«Мои Финансы»** — это современное Android-приложение для эффективного управления личным бюджетом. Основная особенность приложения — **полная поддержка мультивалютности**. Пользователь может записывать расходы в разных валютах, а приложение автоматически консолидирует их, используя актуальные данные о курсах.

Проект выполнен с использованием актуального стека технологий Android-разработки и архитектурного паттерна **MVVM**, что обеспечивает стабильность и удобство масштабирования.

## Ключевые возможности
* **Мультивалютный учет:** Добавление расходов в **BYN** или **USD** с мгновенным отображением в общем списке.
* **Умная аналитика:** Динамическое переключение всей статистики между валютами. При выборе валюты приложение пересчитывает все записи по текущему курсу.
* **Автоматизация курсов (Retrofit):** Получение свежих курсов валют через стороннее API при каждом запуске.
* **Офлайн-режим:** Если интернет недоступен, приложение использует предустановленный или последний сохраненный курс, обеспечивая бесперебойную работу.
* **Персонализация:** * Смена светлой и темной тем (с сохранением выбора после перезапуска).
    * Динамическая локализация (Русский / English).
* **Надежное хранение (Room):** Все данные сохраняются локально в базе данных SQLite.

## Технологический стек
* **Language:** Kotlin
* **Architecture:** MVVM (Model-View-ViewModel)
* **Database:** Room (SQLite)
* **Networking:** Retrofit 2 + Gson
* **Concurrency:** Kotlin Coroutines & Flow
* **UI:** Material Design 3, ViewBinding, Bottom Navigation
* **State Management:** LiveData

## Установка
Для локального запуска проекта:
1.  Убедитесь, что установлена **Android Studio (Ladybug 2024.2.1 или новее)**.
2.  Склонируйте репозиторий:
    ```bash
    git clone [https://github.com/fpmi-pmvs2026/pmvs10a-project-dream_team.git](https://github.com/fpmi-pmvs2026/pmvs10a-project-dream_team.git)
    ```
3.  Откройте папку проекта в Android Studio.
4.  Дождитесь завершения **Gradle Sync**.
5.  Запустите приложение на эмуляторе или физическом устройстве (Shift + F10).

## Использование
1.  **Главная (Home):** Введите название, сумму и выберите валюту (BYN/USD). Расход сохранится в базе и отобразится в ленте.
2.  **Аналитика (Stats):** Нажмите на переключатель **BYN/USD** в верхней части экрана. Вся статистика (общая сумма и категории) мгновенно пересчитается.
3.  **Настройки (Settings):** Используйте переключатель для активации темной темы или кнопки смены языка. Приложение запомнит ваш выбор для следующего сеанса.
