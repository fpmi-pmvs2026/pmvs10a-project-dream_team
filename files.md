# Функциональные требования

Приложение «Мои Финансы» реализует следующие возможности:

### Основные функции
1. **Учет расходов:** Добавление записей с указанием суммы, категории (еда, транспорт и т.д.) и валюты.
2. **Мультивалютность:** Возможность ввода данных в BYN и USD.
3. **Аналитика:** Просмотр общей суммы затрат и детализации по категориям.
4. **Конвертация:** Динамический пересчет всей статистики при переключении между валютами.
5. **Настройки:** Смена темы оформления (светлая/темная) и языка интерфейса (RU/EN).

### Сценарии использования (Use Cases)
* **Пользователь** вводит данные о покупке -> Данные сохраняются в локальную базу данных -> Список обновляется мгновенно.
* **Пользователь** меняет валюту в разделе статистики -> Система запрашивает актуальный курс через API (или берет из кэша) -> Все суммы пересчитываются.
