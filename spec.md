# Дополнительная спецификация

### Ограничения
* Минимальная версия ОС: Android 7.0 (API 24).
* Требуется интернет для получения актуальных курсов валют.

### Надежность
* **Офлайн-режим:** При отсутствии сети используются последние сохраненные курсы валют.
* **Сохранение состояния:** Настройки темы и языка сохраняются в `SharedPreferences` и не сбрасываются после закрытия приложения.

### Безопасность
* Приложение не требует регистрации.
* Все финансовые данные пользователя хранятся исключительно локально на устройстве (в БД SQLite через Room).
